# Ovelin Mall — Android APK (WebView)

تطبيق Android يعرض موقع Ovelin Mall مباشرةً داخل WebView.

## 📥 تحميل التطبيق

اذهب إلى [Releases](../../releases) وحمّل آخر ملف `.apk`.

## 🔄 التحديثات التلقائية

أي تعديل على الموقع `https://ovelinmall-ovelin-mall.hf.space/` يظهر فوراً في التطبيق بدون الحاجة لتحديث الـ APK.

## ⚙️ البناء التلقائي

كل push على `main` يبني APK جديداً تلقائياً عبر GitHub Actions.
