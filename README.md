# Ovelin Mall Android App

تطبيق Android أصيل لمتجر Ovelin Mall الرقمي.

## 📱 تحميل التطبيق

اذهب إلى [Releases](../../releases) وحمّل آخر إصدار من ملف `.apk`.

## 🏗️ بناء التطبيق

يتم بناء الـ APK تلقائياً عند كل push على `main` عبر GitHub Actions.

## 🔗 السيرفر

يتصل التطبيق بـ: `https://ovelinmall-ovelin-mall.hf.space`

## 📋 الصفحات

- الرئيسية — عروض وتصنيفات ومنتجات
- المنتجات — تفاصيل وطلب مباشر
- الطلبات — سجل كامل مع تتبع الحالة
- المحفظة — رصيد وإيداع ومعاملات
- الدعم — تذاكر دعم فني
- الحساب — ملف شخصي وإحصائيات
- الإشعارات — تنبيهات فورية
- الإحالة — برنامج مكافآت
- التحويلات — تحويل رصيد بين المستخدمين
- العروض المحدودة — فلاش سيل

## ⚙️ متطلبات التطوير

- Node.js 20+
- Expo CLI
- Android Studio (للتشغيل المحلي)
